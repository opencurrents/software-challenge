Requirements to run the Rock Paper Scissor game:
	Python 3.x
	Terminal or command line

Playing the Game:
1. Run the game file "rps.py" using comamnd "python rps.py"
2. Then you will enter the Rock, Paper and Scissor game with 2 players which is totally interactive.
3. This game also maintains a score for the games that both the players played.
4. Once you have exited the RPS game you will have option to play the higher versions of the game like RPS-5, RPS-25
   and RPS-101 which works as expected. Please give it a try.
